🔧 One-Time Firebase Setup (Free, 5 minutes)

This app uses Firebase Realtime Database as the signaling server — it's completely free, reliable, and has no ghost-session problems. You only set it up once. Video streams peer-to-peer (not through Firebase) so it's fast.

1
Go to console.firebase.google.com → click Add project → enter any name → click through (no Analytics needed).
2
In your project, click Build → Realtime Database → Create Database → Start in test mode → Enable.
3
Copy the database URL shown (looks like https://YOUR-PROJECT-default-rtdb.firebaseio.com).
4
Paste the URL below and tap SAVE. That's it — all devices must use the same URL.
No database URL saved yet.

⚡ How to Use

1
Complete Firebase setup above first (one-time, 5 min).
2
Receiver: Open URL → stay on RECEIVER tab. Green SERVER dot = ready.
3
Sender: Open same URL → SENDER tab → ▶ START CAMERA + MIC → allow both permissions.
4
Camera appears on receiver within ~3 seconds. Add as many senders as you want — each gets its own tile.
5
Tap ⟳ FLIP to switch front/back camera. Tap 🔇 to unmute audio (muted by default).
⚠ Same Firebase URL must be saved on ALL devices (sender and receiver). Save it in the box above on each device.
