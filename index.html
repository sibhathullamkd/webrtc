How to Use
1
Open this page on both phones — same URL (host it once, open everywhere)
2
Phone A (old phone): tap "Phone A · Send Camera", then "Allow Camera & Start"
3
Phone A: copy the generated offer code (big text box)
4
Phone B (your main phone): tap "Phone B · View Stream", paste the offer code, tap "Generate Answer"
5
Phone B: copy the answer code that appears
6
Phone A: paste the answer code into the second box, tap "Connect" — stream goes live! 🎉
